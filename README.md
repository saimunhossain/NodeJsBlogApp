# NodeJsBlog

This is simple experimental blog application using NodeJS, Express Framework & MongoDB. I have been trying to learn nodeJS, as an experiment I have done this project to know how things work actually in nodeJS.

## Installation

```git clone https://github.com/saimunhossain/NodeJsBlogApp```

```cd NodeJsBlogApp```

```npm install```


*Setup project database, connect to MongoDB & setup all others credentials on .env file* 


```npm start```
